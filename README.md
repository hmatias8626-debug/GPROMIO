# GPROMIO - GPRO Manager Tools

App en Streamlit para calcular combustible de Q2, combustible total, estrategia de paradas y estimación inicial de neumáticos.

## Ejecutar local

```bash
pip install -r requirements.txt
streamlit run app.py
```
